# Advanced Screen

## My equivalent keys
Lost or compromised keys can't be recovered. But they can be replaced.
Your replaced keys remain associated with your identity so that those who trusted or followed those keys still follow you, recognize you You retain your identity even as you rotate through keys.
Manage your replaced keys [here].
- Show all keys that my key has referenced as a subject using the "replace" verb.
  - display with a crossed out green key icon (similar to revoked delegate but green, not blue)
- allow
  - clear
  - edit comment
  - the remain replaced with revokeAt: <since always>, don't even show that. But do include that in updates.

## Replace my key
Create and start using a new key.
No one will know it's you unless you have folks vouch for you all over again. Once one or two do, the others will be eventually notified. But it's still a hassle. Avoid it if you can.

- create and start using a new key
